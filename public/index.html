Documentation API de Films

Cette API vous permet de consulter, d'ajouter, de modifier et de supprimer des films, ainsi que de gérer les inscriptions et les connexions des utilisateurs dans la base de données.

Point d'accès

Récupérer tous les films

  • GET
  • URL: /api/films
  • Paramètres URL: ordre (optionnel), tri (optionnel), Ex: /api/films?tri=annee&ordre=asc
  • Réponse: Liste des films.

Récupérer un film par ID

  • GET
  • URL: /api/films/:id
  • Réponse: Détails du film demandé selon son id.

Ajouter un film

  • POST
  • URL: /api/films
  • Corps de la requête: 
    {
        "titre": "Alien - Le 8ème passager",
        "genres": ["Horreur", "Science-fiction"],
        "description": "Texte",
        "annee": "1979",
        "realisation": "Ridley Scott",
        "titreVignette": "alienle8emepassager.jpg",
        "commentaires": "Texte"
    }
  • Réponse: { message : 'Le film a été ajouté', id: newFilm.id, donnees: film}

Modifier un film

  • PUT
  • URL: /api/films/:id
  • Corps de la requête: Attributs du film à modifier. 
    Ex :
        {
            "genres": ["Drame", "Science-fiction"],
            "annee": "1980"
        }
  • Réponse: { message : 'Le film a été modifié' }

Supprimer un film

  • DELETE
  • URL: /api/films/:id
  • Réponse: { message : 'Le film a été supprimée' }

Authentification et gestion des utilisateurs

Inscription d'un utilisateur

  • POST
  • URL: /api/utilisateurs/inscription
  • Corps de la requête: 
    {
        "courriel": "exemple@domaine.com",
        "mdp": "MotDePasse#1234"
    }
  • Réponse: { message: "L'utilisateur a été ajouté", newUtilisateur: {courriel: "banane@email.com"} }

Connexion d'un utilisateur

  • POST
  • URL: /api/utilisateurs/connexion
  • Corps de la requête: 
    {
        "courriel": "exemple@domaine.com",
        "mdp": "MotDePasse#1234"
    }
  • Réponse: { message: "Vous êtes connecté", utilisateurAValider: { courriel: "banane@email.com" } }
